Phase C — Preprocessing (Doc3 Process)
4

🧹 VIF Multicollinearity Screening

Following Doc3: iterative Variance Inflation Factor (VIF) removal. Compute VIF for all 36 parameters → remove highest VIF if > 10 → repeat until all VIF < 10. VIF chosen over PCA/LASSO to retain physical interpretability of fire drivers.

VIFj = 1 / (1 − R²j)   →   Remove highest if VIF > 10   →   Repeat until all pass
36 → ~24–28 expected after VIF VIF threshold = 10 Preserves physical meaning
5

✂️ Feature Scaling — StandardScaler

Following Doc3: normalize all features to zero mean, unit variance. Fitted on training set ONLY — same transform applied to test set. Critical for SVM (distance-based) and MLP (gradient-based).

x' = (x − μtrain) / σtrain
Fit on Blocks 1–3 only Transform Block 4 with same μ, σ No leakage
6

📋 Train / Test Split

Temporal block split — Blocks 1–3 (2003–2020) = Training, Block 4 (2021–2025) = Test. 10-fold cross-validation on training set for generating out-of-fold meta-features (Doc3 process).

🔵 Train: 30,000 samples (Blocks 1–3)
🔴 Test: 9,249 samples (Block 4)
📋 10-Fold CV for meta-features
Phase D — BIPE: Bi-Layer Stacking Ensemble (Doc3 Architecture)
7

🏗️ BIPE Architecture — SVM + MLP + XGBoost → LR

Doc3's exact architecture applied to our Western Ghats data: three diverse base learners generate probability predictions via 10-fold cross-validation. A Logistic Regression meta-learner combines them — simple, interpretable, no overfitting risk.

Input: VIF-screened features from 30,000 Western Ghats samples
⬇️
Layer 1 — Three Base Learners (10-Fold CV)
🎯 SVM (RBF)
C=1, γ=auto, prob=True
🧠 MLP
100→50, ReLU, Adam
⚡ XGBoost
100 trees, depth=10
▼   ▼   ▼
Out-of-Fold Probabilities → Meta-Features [P̂SVM ∥ P̂MLP ∥ P̂XGB]
▼ ▼
Layer 2 — Meta-Learner
🏆 Logistic Regression
→ Final: P̂(fire | x) ∈ [0, 1]
Pfire = σ(α₁ · P̂SVM + α₂ · P̂MLP + α₃ · P̂XGB + b)    where σ = sigmoid
LR weights show each model's contribution Convex → stable single optimum Simple meta prevents overfitting
8

🔄 10-Fold CV Meta-Feature Generation

For each fold k = 1…10 of the 30,000 training samples:
① Train SVM, MLP, XGBoost on folds ≠ k
② Predict probabilities on held-out fold k
③ Store predictions as meta-features

After all 10 folds → every training sample has unbiased out-of-fold predictions from all 3 models. Train LR meta-learner on these 3 meta-feature columns. Zero data leakage.

30K × 10 folds × 3 models Unbiased OOF predictions Doc3 exact process
